在 Ubuntu 等 Linux 发行版中,从官网下载钉钉 deb 包安装后,点击图标可能无反应,无法启动。本文提供两种有效的解决方案。

## 方案一：使用 patchelf 工具修复

这是推荐的解决方案,适用于 Ubuntu 及其衍生发行版。

```shell
sudo apt install patchelf --fix-missing
cd /opt/apps/com.alibabainc.dingtalk/files/7.8.15-Release.5102301
# 注意：将 "7.8.15-Release.5102301" 替换为你的钉钉实际版本号
sudo patchelf --clear-execstack ./dingtalk_dll.so
```

## 方案二：安装星火商店版本

> [!NOTE]
>
> 请参阅 [星火商店官网](https://www.spark-app.store/)

按照官网指引下载并安装星火商店的 deb 包,然后执行以下命令：

```shell
sudo aptss update --fix-missing
sudo aptss install com.alibabainc.dingtalk
```

### 兼容性说明

> [!WARNING]
>
> **Ubuntu 25.04+ 用户请注意：**由于与 [AppArmor](https://www.apparmor.net/) 安全模块的扩展功能冲突,星火商店提供的 [APM 琥珀运行环境](https://bbs.spark-app.store/d/2221-xing-huo-apmhu-po-bao-qi-fa-yu-ling-long-sha-shou-de-rong-qi-bao-guan-li-demo)可能无法正常使用。

**推荐使用场景：**
- **Ubuntu 及衍生发行版**：优先使用方案一（patchelf）
- **其他 Debian 系发行版**：可使用星火商店获得更便捷的体验
- **AOSC OS**：由于软件仓库完全独立构建,与 Debian 差异较大,星火商店暂不兼容（截至发稿时）

更多信息请访问 [AOSC 官网](https://aosc.io/)。
